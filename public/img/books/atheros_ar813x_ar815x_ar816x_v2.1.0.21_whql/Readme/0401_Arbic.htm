	1  نظرة عامة



تشتمل حزمة تثبيت برنامج التشغيل على Qualcomm Atheros AR81Family جيجابت/برنامج تشغيل وحدة تحكم إيثرنت سريعة.

يشرح ملف README هذا طريقة تثبيت برنامج التشغيل وحزمة التثبيت على جهاز يعمل بإحدى أنظمة التشغيل التالية:

Windows XP وWindows Server 2003 وWindows XP x64 وWindows Server 2003 x64 وWindows Vista وWindows Vista x64 وWindows 7 وWindows 7 x64 وWindows 8 وWindows 8 x64





	2  التثبيت



سيقوم نظام تشغيل Windows تلقائيًا باكتشاف المهايئ. قم بإلغاء كافة مربعات الحوار لأن نظام التشغيل يحاول تثبيت
برنامج التشغيل تلقائيًا.  


ملاحظة: يستطيع المستخدم الدخول باستخدام حقوق إدارية لكي يكون قادرًا على تثبيت برنامج التشغيل.

لتثبيت برنامج التشغيل، تابع كما يلي:

1. قم بتنفيذ ملف تثبيت "Setup.exe".
2. اتبع التعليمات المعروضة أثناء إجراء التثبيت.





	3  تثبيت صامت



لتثبيت برنامج التشغيل بشكل صامت، يجب تنفيذ الأوامر التالية.

setup.exe /s




	4  إلغاء التثبيت



يجب اتباع الخطوات التالية لإلغاء تثبيت برنامج التشغيل.


إزالة برنامج التشغيل

1. انتقل إلى "بدء" > "لوحة التحكم" > "إضافة أو إزالة البرامج".
2. ابحث عن "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver"، حددها وانقر فوق الزر "إزالة".

للحصول على المزيد من المعلومات يرجى الاتصال بالدعم.
