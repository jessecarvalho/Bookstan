	1  概要



ドライバ インストール パッケージには、Qualcomm Atheros AR81Family Gigabit/Fast Ethernet コントローラのドライバが含まれます。

この README ファイルでは、以下のいずれかのオペレーション システムを実行するマシンにドライバとインストール パッケージをインストールする方法について説明します。

Windows XP、Windows Server 2003、Windows XP x64、Windows Server 2003 x64、Windows Vista、Windows Vista x64、Windows 7、Windows 7 x64、Windows 8、Windows 8 x64





	2  インストール



Windows オペレーション システムでは、アダプタは自動的に検出されます。 オペレーション システムがドライバの自動インストールを試みるすべてのダイアログ ボックス 
をキャンセルしてください。  


注:  ドライバをインストールする場合は、管理者権限でログインする必要があります。

ドライバをインストールするには、以下の手順に従います。

1. "Setup.exe" というセットアップ ファイルを実行します。
2. インストール手順中に表示される指示に従ってください。





	3  サイレント インストール



ドライバをサイレント インストールする場合は、次のコマンドを実行してください。

setup.exe /s




	4  アンインストール



ドライバをアンインストールする場合は、以下の手順を実行してください。


ドライバの削除

1. [スタート] > [コントロール パネル] > [プログラムの追加と削除] の順にクリックします。
2. "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver" を探して選択し、[削除] ボタンをクリックします。

詳細については、当社のサポートに連絡してください。
