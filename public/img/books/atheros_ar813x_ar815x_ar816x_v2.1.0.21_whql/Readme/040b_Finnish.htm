		1  Yleistä
	



Ohjaimen asennuspaketti sisältää Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver -ohjaimen.

Tässä README-tiedostossa kerrotaan, kuinka ohjain ja asennuspaketti asennetaan tietokoneeseen, jossa on jokin seuraavista käyttöjärjestelmistä:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64.





	
		2  Asentaminen
	



Windows-käyttöjärjestelmä tunnistaa verkkokortin automaattisesti. Peruuta kaikki valintaikkunat, joissa käyttöjärjestelmä yrittää asentaa 
ohjaimen automaattisesti.  


HUOM. Käyttäjän täytyy kirjautua sisään järjestelmänvalvojan oikeuksin voidakseen asentaa ohjaimen.

Asenna ohjain seuraavasti:

1. Suorita asennustiedosto Setup.exe.
2. Noudata asennuksen aikana näyttöön tulevia ohjeita.





	
		3  Valvomaton asentaminen
	



Jos haluat asentaa ohjaimen valvomatta, suorita seuraavat komennot.

setup.exe /s




	
		4  Asennuksen poistaminen
	



Jos haluat poistaa ohjaimen asennuksen, toimi seuraavalla tavalla.


Ohjaimen poisto

1. Valitse Käynnistä > Ohjauspaneeli > Lisää tai poista sovellus.
2. Etsi Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver, valitse se ja napsauta Poista-painiketta.

Lisätietoja saat tukipalvelustamme.
