Windows Vista, Windows Vista x64

Bu dosyanın içeriği

    1  Genel Bakış
    2  Kurulum
    3  Sessiz Kurulum
    4  Kaldırma 





	
		1  Genel Bakış
	



Sürücü kurulum paketinde Qualcomm Atheros AR81Family Gigabit/Hızlı Ethernet Denetleyicisi Sürücüsü bulunur.

Bu BENİOKU dosyası, sürücünün ve kurulum paketinin aşağıdaki işletim sistemlerini çalıştıran bir bilgisayarda nasıl kurulduğunu açıklar:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	
		2  Kurulum
	



Windows işletim sistemi bağdaştırıcıyı otomatik olarak algılar. İşletim sisteminin sürücüyü otomatik olarak kurmaya çalıştığı tüm iletişim kutularını iptal edin.  


NOT: Kullanıcının sürücüyü kurabilmesi için yönetici olarak oturum açması gerekir.

Sürücüyü kurmak için aşağıdaki adımları takip edin:

1. "Setup.exe" adlı kurulum dosyasını çalıştırın.
2. Kurulum yordamı sırasında ekrandaki yönergeleri takip edin.





	
		3  Sessiz Kurulum
	



Sürücüyü sessiz olarak yüklemek için aşağıdaki komutlar çalıştırılmalıdır.

setup.exe /s




	
		4  Kaldırma
	



Sürücüyü sessiz olarak kaldırmak için aşağıdaki yordam takip edilmelidir.


Sürücünün Kaldırılması

1. "Başlat" > "Denetim Masası" > "Program Ekle veya Kaldır" seçeneklerini belirleyin.
2. "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver" öğesini seçin ve "Kaldır" düğmesini tıklayın.

Ayrıntılı bilgi için lütfen destek hattımızla iletişime geçin.
