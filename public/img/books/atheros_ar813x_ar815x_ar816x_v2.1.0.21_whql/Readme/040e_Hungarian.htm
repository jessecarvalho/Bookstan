		1  Áttekintés
	



A telepítőcsomag tartalmazza az Qualcomm Atheros AR81Family Gigabit/Fast Ethernet vezérlő illesztőprogramját.

Ez a README fájl ismerteti, hogy hogyan kell telepíteni az illesztőprogramot és a telepítőcsomagot olyan gépre, amelyen a következő operációs rendszerek egyike fut:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	
		2  Telepítés
	



A Windows operációs rendszer automatikusan észleli az adaptert. Lépjen ki minden párbeszédpanelből, amelyet az operációs rendszer az illesztőprogram automatikus telepítéséhez megnyit.  


MEGJEGYZÉS: Az illesztőprogram telepítéséhez a felhasználónak rendszergazdaként kell bejelentkeznie.

Az illesztőprogram a következő lépéseket követve telepíthető:

1. Indítsa el a „Setup.exe” fájlt.
2. Kövesse a telepítés során megjelenő utasításokat.





	
		3  Csendes telepítés
	



Ha csendes telepítést szeretne, végre kell hajtani a következő parancsokat.

setup.exe /s




	
		4  Eltávolítás
	



Az illesztőprogram a következő módon távolítható el.


Az illesztőprogram eltávolítása

1. Válassza a „Start” > „Vezérlőpult” > „Programok telepítése és törlése” parancsot.
2. Keresse meg az „Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver” elemet, jelölje ki, és kattintson az „Eltávolítás” gombra.

További tájékoztatásért forduljon ügyfélszolgálatunkhoz.
