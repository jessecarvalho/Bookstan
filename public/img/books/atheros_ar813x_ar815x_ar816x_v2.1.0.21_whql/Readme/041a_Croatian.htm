	1  Pregled



Paket za instaliranje pogonskog programa sadrži pogonski program Qualcomm Atheros AR81Family Gigabit/brzi kontroler Etherneta.

Ova datoteka README objašnjava kako instalirati pogonski program i paket za instalaciju na aparat koji pokreće jedan od sljedećih operacijskih sustava:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Instalacija



Operacijski sustav Windows će automatski otkriti adapter. Poništite sve dijaloške kutije kojim operacijski sustav pokušava instalirati
pogonski program automatski.  


NAPOMENA: Korisnik se mora prijaviti kao administrator kako bi mogao instalirati pogonski program.

Kako bi instalirali pogonski program, nastavite sljedećim koracima:

1. Izvršite datoteku za podešavanje "Setup.exe".
2. Slijedite upute koje se prikazuju tijekom postupka instalacije.





	3  Tiha instalacija



Kako bi tiho instalirali pogonski program moraju biti izvršene sljedeće naredbe.

setup.exe /s




	4  Deinstalacija



Kako bi deinstalirali pogonski program morate slijediti sljedeći postupak.


Odstranjenje pogonskog programa

1. idite na "Start" > "Control Panel" > "Add or Remove Programs" ("Pokreni" > "Upravljačka ploča" > "Dodaj ili ukloni programe").
2. Pronađite "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", odaberite ga te kliknite gumb "Remove" ("Ukloni").

Za više informacija molimo kontaktirajte našu podršku.
