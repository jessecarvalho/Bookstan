Αρχείο Readme για τα πακέτα εγκατάστασης AR81Family														

Υποστήριξη: Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller					
	
Λειτουργικά συστήματα:																										
Windows XP, Windows Server 2003				                  
Windows XP x64, Windows Server 2003 x64                               
Windows Vista, Windows Vista x64	
Windows 7, Windows 7 x64																				
Windows 8, Windows 8 x64

Το παρόν αρχείο περιλαμβάνει

    1  Επισκόπηση
    2  Εγκατάσταση
    3  Σιωπηρή εγκατάσταση
    4  Κατάργηση εγκατάστασης 





	
		1  Επισκόπηση
	



Το πακέτο εγκατάστασης προγράμματος οδήγησης περιλαμβάνει το πρόγραμμα οδήγησης Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller.

Το παρόν αρχέιο README εξηγεί τον τρόπο εγκατάστασης του προγράμματος οδήγησης και του πακέτου εγκατάστασης σε συσκευή με ένα από τα ακόλουθα λειτουργικά συστήματα:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	
		2  Εγκατάσταση
	



Το λειτουργικό σύστημα Windows ανιχνεύει αυτόματα τον προσαρμογέα. Ακυρώστε όλα τα παράθυρα διαλόγου κατά την απόπειρα αυτόματης εγκατάστασης του προγράμματος οδήγησης 
από το λειτουργικό σύστημα.  


ΣΗΜΕΙΩΣΗ: Για την εγκατάσταση του προγράμματος οδήγησης, ο χρήστης πρέπει να συνδεθεί με δικαιώματα διαχειριστή.

Για την εγκατάσταση του προγράμματος οδήγησης, ακολουθήστε τα παρακάτω βήματα:

1. Εκτελέστε το αρχείο εγκατάστασης "Setup.exe".
2. Ακολουθήστε τις οδηγίες που εμφανίζονται κατά τη διαδικασία εγκατάστασης.





	
		3  Σιωπηρή εγκατάσταση
	



Για τη σιωπηρή εγκατάσταση του προγράμματος οδήγησης, πρέπει να εκτελεστούν οι ακόλουθες εντολές.

setup.exe /s




	
		4  Κατάργηση εγκατάστασης
	



Για την κατάργηση εγκατάστασης του προγράμματος οδήγησης, πρέπει να τηρηθούν οι παρακάτω διαδικασίες.


Κατάργηση προγράμματος οδήγησης

1. Μεταβείτε στο μενού "Έναρξη" > "Πίνακας ελέγχου" > "Προσθαφαίρεση προγραμμάτων".
2. Εντοπίστε το πρόγραμμα οδήγησης "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", επιλέξτε το και πατήστε το κουμπί "Κατάργηση".

Για περισσότερες πληροφορίες, επικοινωήστε με το τμήμα υποστήριξης.
