	1  Présentation



Le programme d'installation du pilote contient Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver.

Ce fichier LISEZ-MOI explique comment installer le pilote et le programme d'installation sur un ordinateur exécutant un des systèmes d'exploitation suivants :

Windows XP, Windows Server 2003, Windows XP Édition x64, Windows Server 2003 Édition x64, Windows Vista, Windows Vista Édition x64, Windows 7, Windows 7 Édition x64, Windows 8, Windows 8 Édition x64





	2  Installation



Le système d'exploitation Windows détecte automatiquement la carte. Cliquez sur Annuler dans toutes les boîtes de dialogue que le système d'exploitation propose pour installer 
le pilote automatiquement.  


REMARQUE : L'utilisateur qui se connecte doit disposer des droits d'administrateur pour être en mesure d'installer le pilote.

Pour installer le pilote, effectuez les opérations suivantes :

1. Exécutez le fichier d'installation "Setup.exe".
2. Suivez les instructions qui s'affichent pendant la procédure d'installation.





	3  Installation automatisée



Pour installer le pilote de façon automatique, exécutez les commandes suivantes.

setup.exe /s




	4  Désinstallation



Pour désinstaller le pilote, exécutez les opérations suivantes.


Suppression du pilote

1. Cliquez sur "Démarrer" > "Panneau de configuration" > "Ajouter ou supprimer des programmes".
2. Localisez le programme "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", sélectionnez-le, puis cliquez sur le bouton "Supprimer".

Pour plus d'informations, prenez contact avec notre support technique.
