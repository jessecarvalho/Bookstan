		1  Обзор
	



Пакет установки драйвера содержит драйвер Gigabit/Fast Ethernet-контроллера Qualcomm Atheros AR81Family.

Данный файл README содержит инструкции по установке драйвера и пакета установки на компьютере под управлением одной из следующих операционных систем:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	
		2  Установка
	



Операционная система Windows обнаружит адаптер автоматически. Щелкайте кнопку отмены во всех диалоговых окнах, открываемых 
для автоматической установки драйвера.  


ПРИМЕЧАНИЕ. Для установки драйвера пользователь должен выполнить вход в систему с правами администратора.

Порядок установки драйвера:

1. Запустите файл установки «Setup.exe».
2. Следуйте отображаемым инструкциям.





	
		3  Автоматическая установка
	



Чтобы установить драйвер в автоматическом режиме, необходимо выполнить следующую команду:

setup.exe /s




	
		4  Удаление
	



Чтобы удалить драйвер, необходимо выполнить указанные ниже действия.


Удаление драйвера

1. Выберите «Пуск» > «Панель управления» > «Установка и удаление программ».
2. Найдите устройство «Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver», выберите его и щелкните кнопку «Удалить».

Для получения дополнительной информации обратитесь в нашу службу поддержки.
