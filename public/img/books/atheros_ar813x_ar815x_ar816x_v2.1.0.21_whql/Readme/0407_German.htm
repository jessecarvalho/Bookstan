	1  Übersicht



Das Treiberinstallationspaket enthält den Treiber für den Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller.

In dieser README-Datei wird der Installationsprozess für Treiber und Installationspaket auf einem Computer unter einem der folgenden Betriebssysteme beschrieben:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Installation



Der Adapter wird vom Windows-Betriebssystem automatisch erkannt. Beenden Sie alle Dialogfelder, die besagen, dass das Betriebssystem den Treiber 
automatisch installieren möchte.  


HINWEIS: Benutzer müssen sich mit Administratorrechten anmelden, um den Treiber zu installieren.

Befolgen Sie zur Installation des Treibers die folgenden Schritte:

1. Führen Sie die Datei "Setup.exe" aus.
2. Befolgen Sie die Installationsanweisungen.





	3  Unbeaufsichtigte Installation



Zur unbeaufsichtigten Installation des Treibers müssen die folgenden Befehle ausgeführt werden:

setup.exe /s




	4  Deinstallation



Zur Deinstallation des Treibers führen Sie die folgenden Schritte aus:


Entfernen des Treibers

1. Wählen Sie die Befehlsfolge "Start" > "Systemsteuerung" > "Software".
2. Wählen Sie "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver" aus, und klicken Sie auf die Schaltfläche "Entfernen".

Bitte wenden Sie sich für weitere Informationen an unseren Support.
