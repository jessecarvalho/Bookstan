	1  概述



该驱动程序安装包中包含 Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver。

此 README 文件说明如何在运行下列操作系统之一的计算机上安装该驱动程序和安装包：

Windows XP、Windows Server 2003、Windows XP x64、Windows Server 2003 x64、Windows Vista、Windows Vista x64、Windows 7、Windows 7 x64、Windows 8、Windows 8 x64	





	2  安装



Windows 操作系统会自动检测适配器。 取消提示操作系统将试图自动安装 
驱动程序的所有对话框。  


注意： 用户必须以具有管理权限的帐户登录，才能安装驱动程序。

若要安装驱动程序，请按照下列步骤进行操作：

1. 执行安装文件"Setup.exe"。
2. 按照安装过程中显示的说明进行操作。





	3  无提示安装



若要在不出现任何无提示的情况下安装驱动程序，请执行下列命令。

setup.exe /s




	4  卸载



若要卸载驱动程序，请执行下列步骤。


删除驱动程序

1. 转到"开始">"控制面板">"添加或删除程序"。
2. 找到"Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver"，选中后单击"删除"按钮。

有关更多信息，请与我们的支持部门联系。
