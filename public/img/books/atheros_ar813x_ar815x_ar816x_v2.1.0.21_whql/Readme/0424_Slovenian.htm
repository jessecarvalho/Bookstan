	1  Pregled



Ta gonilniški namestitveni paket vsebuje gonilnik za Qualcomm Atheros AR81Family Gigabit/Hitri Ethernet krmilnik.

Ta README datoteka razlaga način namestitve gonilnika in namestitvenega paketa na računalnikih z enim od naslednjih operacijskih sistemov:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Namestitev



Windows operacijski sistem bo samodejno zaznal adapter. Prekličite vsa pogovorna okna, ki jih operacijski sistem odpre, za samodejno
namestitev gonilnika.  


OPOMBA: Da bi lahko namestil gonilnik, se uporabnik mora prijaviti z administratorskimi pravicami.

Za namestitev gonilnika, postopajte kot sledi:

1. Zaženite postavitveno datoteko "Setup.exe".
2. Sledite navodilom, ki se prikazujejo med postopkom namestitve.





	3  Tiha namestitev



Za tiho namestitev gonilnika je treba izvesti naslednje ukaze.

setup.exe /s




	4  Odstranitev



Za odstranjevanje gonilnika je treba izvesti naslednji postopek.


Odstranitev gonilnika

1. Odprite "Start" > "Control Panel" (Kontrolna plošča) > "Add or Remove Programs" (Dodaj ali odstrani programe).
2. Poiščite "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", ga izberite in kliknite gumb "Remove" (Odstrani).

Za več informacij se prosimo obrnite na našo službo podpore.
