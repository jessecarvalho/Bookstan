		1  ภาพรวม
	



ชุดการติดตั้งไดรเวอร์ประกอบด้วยไดรเวอร์สำหรับ Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller

ไฟล์ README นี้อธิบายวิธีการติดตั้งไดรเวอร์และชุดการติดตั้งบนเครื่องที่ทำงานด้วยระบบปฏิบัติการหนึ่งประเภทต่อไปนี้:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64	





	
		2  การติดตั้ง
	



ระบบปฏิบัติการ Windows จะตรวจพบอะแด็ปเตอร์โดยอัตโนมัติ ยกเลิกกล่องโต้ตอบทั้งหมดที่ระบบปฏิบัติการพยายามจะติดตั้ง 
ไดรเวอร์โดยอัตโนมัติ  


หมายเหตุ: ผู้ใช้ต้องเข้าระบบในฐานะผู้ดูแลระบบเพื่อให้สามารถติดตั้งไดรเวอร์ได้

ในการติดตั้งไดรเวอร์ ทำตามขั้นตอนดังนี้:

1. เรียกทำงานไฟล์ติดตั้ง "Setup.exe"
2. ทำตามคำแนะนำที่แสดงขึ้นระหว่างขั้นตอนการติดตั้ง





	
		3  การติดตั้งโดยไม่ต้องยืนยัน
	



ในการติดตั้งไดรเวอร์โดยผู้ติดตั้งไม่ต้องยืนยัน ให้ใส่คำสั่งกระทำการดังนี้

setup.exe /s




	
		4  การถอนการติดตั้ง
	



ในการถอนการติดตั้งไดรเวอร์ ควรดำเนินการดังนี้


การลบไดรเวอร์

1. ไปที่ "Start" > "Control Panel" > "Add or Remove Programs"
2. ค้นหา "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver" เลือกข้อนั้น และคลิกปุ่ม "Remove" 

สำหรับข้อมูลเพิ่มเติม โปรดติดต่อฝ่ายสนับสนุนของเรา
