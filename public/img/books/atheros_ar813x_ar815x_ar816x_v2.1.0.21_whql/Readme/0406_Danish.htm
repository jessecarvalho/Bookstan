		1  Oversigt
	



Driverinstallationspakken indeholder Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver.

I denne README-fil forklares, hvordan driveren og installationspakken installeres på en computer, der kører et af følgende operativsystemer:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	
		2  Installation
	



Windows-operativsystemet registerer automatisk adapteren. Annuller alle dialogbokse, hvor operativsystemet forsøger at installere 
driveren automatisk.  


BEMÆRK! Brugeren skal logge på med administratorrettigheder for at kunne installere driveren.

Udfør følgende trin for at installere driveren:

1. Kør installationsfilen "Setup.exe".
2. Følg instruktionerne, der vises under installationsproceduren.





	
		3  Uovervåget installation
	



Hvis driveren skal installeres uovervåget, skal følgende kommandoer køres.

setup.exe /s




	
		4  Afinstallation
	



Hvis driveren skal afinstalleres, skal følgende procedure udføres.


Driverfjernelse

1. Gå til "Start" > "Kontrolpanel" > "Tilføj eller fjern programmer".
2. Find "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", marker den, og klik på knappen "Fjern".

Kontakt vores support for at få yderligere oplysninger.
