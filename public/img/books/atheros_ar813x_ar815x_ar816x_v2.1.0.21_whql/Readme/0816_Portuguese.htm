	1  Panorâmica geral



A embalagem de instalação da driver contém o controlador de driver AR81Family Gigabit/Fast Ethernet Qualcomm Atheros.

Este ficheiro LEIA-ME explica como instalar a driver e a embalagem de instalação de uma máquina a decorrer com um dos seguintes sistemas operativos:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Instalação



O sistema operativo Windows irá detectar o adaptador automaticamente. Cancele todas as caixas de diálogo que o sistema operativo tenta instalar
a driver automaticamente.  


NOTA: O utilizador tem que iniciar sessão com direitos administrativos para ser capaz de instalar a driver.

Para instalar a driver proceder como se segue:

1. Execute o ficheiro de setup "Setup.exe".
2. Siga as instruções mostradas durante o processo de instalação.





	3  Instalação Silenciosa



De modo a instalar a driver silenciosamente, deverão ser executados os seguintes comandos.

setup.exe /s




	4  Desinstalação



De modo a desinstalar a driver deverá ser levado a cabo o seguinte procedimento.


Retirar a driver

1. Vá para "Iniciar" > "Painel de Controlo" > "Adicionar ou Remover Programas".
2. Encontre "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", seleccione e clique no botão "Remover".

Para mais informações, contacte o nosso serviço de apoio.
