	1  Panoramica



Il pacchetto di installazione dei driver contiene il driver relativo al controller Gigabit/Fast Ethernet Qualcomm Atheros AR81Family.

Il presente file LEGGIMI illustra la procedura di installazione del driver e del pacchetto di installazione su un computer che esegue uno dei sistemi operativi riportati di seguito:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Installazione



Il sistema operativo Windows rileverà automaticamente l'adattatore. Annullare tutte le finestre di dialogo di installazione automatica del driver.  


NOTA: per installare il driver, l'utente deve eseguire l'accesso con diritti di amministratore.

Per installare il driver, procedere come segue:

1. Eseguire il file di installazione "Setup.exe".
2. Seguire le istruzioni visualizzate durante la procedura di installazione.





	3  Installazione invisibile all'utente



Per installare il driver in maniera invisibile all'utente, eseguire i comandi riportati di seguito.

setup.exe /s




	4  Disinstallazione



Per disinstallare il driver attenersi alla seguente procedura.


Rimozione del driver

1. Selezionare "Start" > "Pannello di controllo" > "Installazione applicazioni".
2. Individuare "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", selezionarlo e fare clic sul pulsante "Rimuovi".

Per ulteriori informazioni rivolgersi al servizio di assistenza.
