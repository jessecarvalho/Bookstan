Fişier Readme pentru pachetul de instalare AR81Family														

Compatibil: Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller					
	
Sisteme de operare:																										
Windows XP, Windows Server 2003				                  
Windows XP x64, Windows Server 2003 x64
Windows Vista, Windows Vista x64
Windows 7, Windows 7 x64
Windows 8, Windows 8 x64

Acest fişier conţine

    1  Prezentare generală
    2  Instalare
    3  Instalare rapidă
    4  Dezinstalare 





	1  Prezentare generală



Pachetul de instalare al driver-elor conţine driver-ul Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller.

Acest fişier README explică modalitatea de instalare a driver-ului şi pachetul de instalare pe un aparat care rulează unul din următoarele sisteme de operare:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Instalare



Sistemul de operare Windows va detecta în mod automat adaptorul. Anulaţi toate căsuţele de dialog prin care sistemul de operare intenţionează să instaleze
driver-ul în mod automat.  


NOTĂ: Utilizatorul trebuie să se autentifice ca administrator pentru a putea instala driver-ul.

Pentru a instala driver-ul, urmaţi instrucţiunile:

1. Executaţi fişierul de instalare "Setup.exe".
2. Urmaţi instrucţiunile afişate în timpul procedurii de instalare.





	3  Instalare rapidă



Pentru a instala în mod rapid driver-ul, trebuie executate următoarele comenzi.

setup.exe /s




	4  Dezinstalare



Pentru a dezinstala driver-ul trebuie urmată procedura de mai jos.


Înlăturare Driver

1. Mergeţi la "Start" > "Control Panel (Panou de control)" > "Add or Remove Programs (Adăugare sau înlăturare programe)".
2. Găsiţi "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", selectaţi-l şi daţi clic pe butonul "Remove (Înlătură)".

Pentru informaţii suplimentare vă rugăm să ne contactaţi pentru suport.
