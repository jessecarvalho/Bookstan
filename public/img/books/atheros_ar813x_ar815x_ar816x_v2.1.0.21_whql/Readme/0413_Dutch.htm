	1  Overzicht



Het installatiepakket voor het stuurprogramma bevat de Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver.

In dit Leesmij-bestand wordt uitgelegd hoe u het stuurprogramma en installatiepakket installeert op een apparaat met een van de volgende besturingssystemen:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Installatie



Het Windows-besturingssysteem detecteert de adapter automatisch. Annuleer alle dialoogvensters die het besturingssysteem automatisch proberen te installeren.  


OPMERKING: de gebruiker moet zich aanmelden als beheerder om het stuurprogramma te kunnen installeren.

Ga als volgt te werk om het stuurprogramma te installeren:

1. Voer het installatiebestand "Installatie.exe" uit.
2. Volg de instructies die tijdens de installatie op het scherm worden weergegeven.





	3  Installatie op de achtergrond



De volgende opdrachten moeten worden uitgevoerd om het stuurprogramma op de achtergrond te kunnen installeren.

setup.exe /s




	4  Installatie verwijderen



Ga als volgt te werk om het stuurprogramma te verwijderen.


Stuurprogramma verwijderen

1. Ga naar "Start" > "Configuratiescherm" > "Software".
2. Zoek "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", selecteer dit en klik op "Verwijderen".

Neem voor meer informatie contact op met de ondersteuning.
