	1  Información general



El paquete de instalación del controlador contiene el Qualcomm Atheros AR81Family Gigabit/Fast Ethernet Controller Driver.

Este archivo LÉAME explica cómo instalar el controlador y el paquete de instalación en una máquina que funcione con alguno de los siguientes sistemas operativos:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Instalación



El sistema operativo de Windows detectará el adaptador automáticamente. Cancele todos los cuadros de diálogo que intenten que el sistema operativo instale 
automáticamente el controlador.  


NOTA: El usuario tiene que iniciar la sesión con derechos de administrador para instalar el controlador.

Para instalar el controlador lleve a cabo los siguientes pasos:

1. Ejecute el archivo de instalación "Setup.exe".
2. Siga las instrucciones que aparecen durante el proceso de instalación.





	3  Instalación silenciosa



Para realizar una instalación silenciosa del controlador, debe ejecutar los siguientes comandos.

setup.exe /s




	4  Desinstalación



Para desinstalar el controlador debe seguir los siguientes pasos.


Eliminación del controlador

1. Vaya a "Inicio" > "Panel de control" > "Agregar o quitar programas".
2. Localice "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", selecciónelo, y haga clic en el botón "Eliminar".

Para obtener más información, póngase en contacto con nuestro servicio técnico.
