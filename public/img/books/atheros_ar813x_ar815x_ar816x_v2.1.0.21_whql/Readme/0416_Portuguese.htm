	1  Descrição-geral



O pacote de instalação do controlador contém o Controlador Qualcomm Atheros AR81Family Gigabit/Fast Ethernet.

Este ficheiro LEIA-ME explica como instalar o controlador e o pacote de instalação num computador com um dos seguintes sistemas operativos instalado:

Windows XP, Windows Server 2003, Windows XP x64, Windows Server 2003 x64, Windows Vista, Windows Vista x64, Windows 7, Windows 7 x64, Windows 8, Windows 8 x64





	2  Instalação



O sistema operativo Windows detecta o adaptador automaticamente. Cancele todas as caixas de diálogo em que o sistema operativo tenta instalar 
o controlador automaticamente.  


NOTA: o utilizador tem de iniciar sessão com direitos administrativos para poder instalar o controlador.

Para instalar o controlador, proceda do seguinte modo:

1. Execute o ficheiro de configuração "Config.exe".
2. Siga as instruções apresentadas durante o processo de instalação.





	3  Instalação Automática



Para instalar o controlador automaticamente, devem ser executados os comandos que se seguem.

setup.exe /s




	4  Desinstalação



Para desinstalar o controlador, deve ser adoptado o procedimento que se segue.


Remoção do Controlador

1. Vá para "Iniciar" > "Painel de Controlo" > "Adicionar ou Remover Programas".
2. Procure a opção "Qualcomm Atheros Communications Inc.(R) AR81Family Gigabit/Fast Ethernet Driver", seleccione-a e clique no botão "Remover".

Para mais informações, contacte o nosso suporte técnico.
